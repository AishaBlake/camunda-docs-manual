Query for the number of historic tasks that fulfill the given parameters.
Takes the same parameters as the [get historic tasks]({{< ref "/reference/rest/history/task/get-task-query.md" >}}) method.
Corresponds to the size of the result set of the [get tasks (POST)]({{< ref "/reference/rest/history/task/post-task-query.md" >}}) method and takes the same parameters.


# Method

POST `/history/task/count`

## Request

POST `/history/task/count`

        [{"name": "varName",
